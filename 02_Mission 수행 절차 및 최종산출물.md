이번 미션은 노코드 자동화 도구를 활용해 반복 업무를 자동화하는 워크플로우를 직접 설계하고 구현하는 것입니다. 미션을 성공적으로 수행하기 위한 절차와 방법, 그리고 최종 산출물을 안내해 드립니다.(NotebookLM 에서 작성)

**1. 미션 수행 절차 및 방법**

- **공통 구현 규칙:**
    - 모든 워크플로우는 문서 설계에 그치지 않고 반드시 실제로 동작해야 합니다.
    - 구현하는 워크플로우에는 **1개 이상의 Trigger(시작 이벤트)**, **2개 이상의 Action(처리 동작)**, **1개 이상의 조건 분기(Filter/Router)**가 포함되어야 합니다.
    - 조건 분기를 설정한 경우, 각 분기 경로가 실제로 1회 이상 실행된 결과를 확인할 수 있도록 테스트해야 합니다.
- **[프로젝트 1] 자동화 도구 비교 구현 방법:**
    - Make, Zapier 등 2개 이상의 서로 다른 자동화 도구를 사용하여 완전히 동일한 구조의 워크플로우를 구현합니다.
    - 구현 후 UI/UX, 설정 난이도, 연동 서비스 범위, 무료 플랜 범위, 실행 로그 확인 방식 등 최소 5개 이상의 항목을 기준으로 도구를 비교 분석합니다.
    - 분석을 바탕으로 각 도구의 장단점과 어떤 상황에서 사용하기 적합한지 의견을 도출합니다.
- **[프로젝트 2] 자유 주제 자동화 설계 방법:**
    - 본인이 자동화하고 싶은 반복 업무 1개를 자유롭게 정의합니다.
    - 해당 업무에 적합한 도구 1개를 선정하고 그 이유를 구체적으로 작성합니다.
    - Trigger 발생 시 자동 실행되는 구조를 구현하고 워크플로우 흐름을 단계별로 설명합니다.
- **수행 시 유의사항:**
    - 화면 캡처나 문서에 API Key, 토큰, 비밀번호 등의 민감정보가 노출되지 않도록 마스킹(***) 처리해야 합니다.
    - 과금 리스크 완화를 위해 무료 플랜으로 완수 가능한 조합을 우선으로 고려하며, 유료 결제가 필요했다면 그 이유와 무료 대안을 보고서에 기재해야 합니다.

_(선택 사항)_ 보너스 과제로 워크플로우에 ChatGPT 등 생성형 AI를 연동해 텍스트를 자동 생성하게 하거나, 오류 발생 시 메신저로 실패 알림을 보내는 전략을 추가해 볼 수 있습니다.

이번 미션을 수행하기 위해 사용할 수 있는 2가지 자동화 도구로 **Make**와 **Zapier**를 추천합니다.

이 플랫폼들은 코딩을 할 필요 없이 **Trigger(시작 이벤트)**와 **Action(처리 동작)**을 시각적으로 연결하여 반복적인 업무를 자동화할 수 있도록 지원합니다.

특히 과제의 제약 사항 중 '과금 리스크 완화 가이드'에서는 무료 플랜으로 완수 가능한 조합을 우선 고려하도록 권장하고 있으며, 그 예시로 **Make의 무료 제공 범위**와 Google Sheets를 결합하여 사용하는 방법을 언급하고 있으니 프로젝트 수행 시 참고하시기 바랍니다

**2. 최종 산출물**

미션을 마치면 다음 2가지 프로젝트에 대한 산출물을 제출해야 합니다.

- **[프로젝트 1] 자동화 도구 비교 구현 산출물**
    - 각 도구별 워크플로우 구성 화면 캡처
    - 실행 결과 화면 캡처
    - **비교 분석 보고서 (Markdown 또는 PDF 형식):** 사용한 도구 이름, 구현 과정 요약, 5개 이상의 비교 항목, 장단점 및 적합한 상황에 대한 의견 포함
- **[프로젝트 2] 자유 주제 자동화 설계 및 구현 산출물**
    - 자동화할 반복 업무 1개 정의 및 선정한 도구 1개에 대한 선정 이유
    - **워크플로우 설계 문서:** 설명 내용 또는 다이어그램 포함
    - 구현 화면 캡처 및 실행 결과 화면 캡처
